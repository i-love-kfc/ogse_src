== Движковые правки ==
+ Увеличенная дальность прорисовки травы
	команда r__detail_radius [49, 250].
	Исправлено мигание травы (тянется еще с оригинальной игры)
+ Расширенные пределы регулировки плотности травы
	команда r__detail_density [0.06, 1.00]
+ Оптимизация рендеринга травы
	команда r2_details_opt [on\off]
	Прорежает траву на расстоянии. При плотности 0.06 и дальности 250 фпс 25 при включенной настройке против 15 при выключенной.
	Применяется при перезагрузке игры.
+ Расширенные пределы регулировки переключения лодов
	команда r__geometry_lod [0.1, 5.0]
+ Расширенные пределы регулировки переключения мип-уровней
	команда r2_tf_mipbias [-3, 3]	
+ Блудмарки
	команда r__bloodmarks [on/off]
+ Увеличенная дальность переключения каскадов теней
	команда r2_sun_near [1.0, 150.0]
	Применять осторожно, чем больше значение - тем больше артефактов из-за ограниченного разрешения карт теней.
+ Правильные тени от солнца
+ Дополнительные разрешения карт теней
	Что это дает: тени имеют более высокую детализацию, что особенно заметно в утренние часы при включенном фиксе правильных теней.
	Также появляется возможность установить более высокий r2_sun_near без угрозы получить полоски на всех поверхностях.
	В 7-8 часов утра игрового времени, когда тени от солнца имеют наименьшую детализацию, максимальное значение r2_sun_near составляет:
		при -smap4096 - 35 
		при -smap8192 - 70 
	Настраиваются через ключи запуска приложения.
	-smap8192 - разрешение 8192x8192.
	Эти разрешения поддерживаются далеко не всеми видеокартами, понятное дело. Разрешение 8192x8192 поддерживается DX10-видеокартами, начиная с GF 8xxx и Radeon HD 2xxx.
	Кроме того, требуется определенное количество видеопамяти - минимум 512 Мб для 8192x8192.
	Эти условия проверяются в движке, если видеоадаптер не поддерживает такой размер текстур, разрешение сбросится на ближайшее подходящее.

== Шейдерные правки ==
+ Саншафты
	команда r2_sunshafts [qt_off/qt_low/qt_medium/qt_high/qt_extreme]
			r2_sunshafts_mode [screen_space/volumetric] - тип саншафтов
	Также управляются из погодных конфигов - интенсивность (sun_shafts) и плотность (sun_shafts_length)
+ Dynamic Depth Of Field
	команды r2_dof [qt_off/qt_low/qt_medium/qt_high/qt_extreme] - включает сам эффект
			r2_dof_zoom [on/off] - включает dof при прицеливании
			r2_dof_reload [on/off] - включает dof при перезарядке
	Применяется bokeh блюр.
+ SSAO
	команды r2_ssao [qt_off/qt_low/qt_medium/qt_high/qt_extreme] - включает сам эффект
			r2_ssao_mode [ssdo/hbao] - переключение режимов
+ Camera-Based Motion Blur
	команды r2_mblur [on/off] - включает сам эффект
			r2_mblur_amount [0.0, 1.0] - степень проявления эффекта
	родной эффект сталкира, немного исправил в связи с фиксами рендера
+ Мягкие края воды	
	команда r2_soft_water [on/off]
	требует перезагрузки игры для применения
+ Мягкие партиклы
	команда r2_soft_particles [on/off]
	требует перезагрузки игры для применения
+ Parallax Occlusion Mapping (ati, допилено sky4ce)
	команда r2_steep_parallax [qt_off/qt_low/qt_medium/qt_high/qt_extreme] - качество
	требует перезагрузки игры для применения
+ Антиалиасинг
	команда r2_aa [qt_off/qt_low/qt_medium/qt_high/qt_extreme]	- качество
			r2_aa_mode	[fxaa/smaa]				- тип антиалиасинга.
+ "Термическое" зрение
	команда r2_render_mode [normal/thermal]
	есть нагрев/охлаждение оружия, добавлен шум в качестве "помех"
+ Мягкие тени (cjahyo)
	команда r2_soft_shadows [qt_off/qt_low/qt_medium/qt_high/qt_extreme] - качество
	требует перезагрузки игры для применения
+ Стекание капель по экрану при дожде
	команда r2_rain_drops [Off/On]
+ Детальный бамп
	команда r2_detail_bump [Off/On]
	требует перезагрузки игры для применения
+ Размытие и затухание теней от spot-источников
	настраивается в accum_spot.ps
	требует перезагрузки игры для применения	
+ Цветокоррекция картинки (color grading)
	r2_color_grading [0.0, 0.0, 0.0, 0.0] - первые три компонента - цвет, последний - интенсивность эффекта
	работает в риал-тайм, можно управлять из скриптов в зависимости от ситуации
+ Блики от ламп
	r2_lens_flare [on/off].
+ Настройка резкости картинки
	r2_image_sharpening [0.0 - 1.0]
+ Отражения (Realtime Local Reflections)
	r2_reflections [qt_off/qt_low/qt_high] - качество
	r2_reflections_mode [water/ground/both] - тип отражений (на воде/на земле/везде)
	Требует перезагрузки игры.
+ Намокание
	r2_wet_surfaces [on/off]
+ Объемный свет ламп
	r2_volumetric_lights [on/off]
	
== Мелочи ==
+ рендертаргеты G-буфера с позициями и нормалью теперь очищаются, чтобы не было ghosting на небе
+ в шейдеры экспортированы три глобальных вектора - ogse_c_resolution, ogse_c_screen, ogse_c_rain. Подробно см. common.h
+ при компиляции шейдеров убран флаг D3DXSHADER_USE_LEGACY_D3DX9_31_DLL. Тем самым отключено использование старого компилятора.
Требует переработки игровых шейдеров для совместимости с компилятором, но зато можно использовать все фичи SM 3.0.
+ внедрен кеш шейдеров. При сворачивании/разворачивании игры в fullscreen задержка картинки на GTS 250 составляет менее 3 сек.
+ внедрены лоды травы. Позволяет сэкономить немного фпс при высокой плотности/дальности отрисовки травы.
